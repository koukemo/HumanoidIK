# Humanoid IK
![Humanoid IK](https://img.shields.io/badge/Humanoid-IK-blue)

Sample Using Inverse Kinematics in Unity.

## Developing

---

### Setting up Dev

Create an Animator controller in the Projects view. <br>
![create-animator-controller](docs/figures/create_animator-controller.png)

Open the animator tab. <br>
![open-animator](docs/figures/open_animator.png)

Enable IK Pass. <br>
![on-IKPass](docs/figures/on_IKPass.png)

Apply controller to the animator of the model you wish to move. <br>
![attach-controller](docs/figures/attach_controller.png)
